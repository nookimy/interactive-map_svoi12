Инструкция по заполнению данных для locationData

Этот файл содержит данные по участкам, которые используются для отображения информации на карте и в балуне.

Общая структура объекта locationData

Каждый участок в locationData задаётся как объект, где ключ — это id элемента SVG, а значение — объект с параметрами:

const locationData = {
  'id_элемента': {
    name: 'Название участка или номер',   // Обязательный параметр
    status: STATUS_FREE,                  // Статус участка (см. ниже)    
    area: 9.45,                           // Площадь участка в сотках (число)
    price: 12345678,                      // Цена участка в рублях (необязательно, может рассчитываться автоматически)
    houseArea: 150,                       // Площадь дома в м² (необязательно)
    housePrice: 23500000,                 // Цена дома в рублях (необязательно, может рассчитываться автоматически)
    totalPrice: 30000000,                 // Общая стоимость участка и дома (необязательно, может рассчитываться автоматически)
    img: 'house-180.jpg',                 // Имя файла изображения дома (необязательно)
    'house-link': 'https://example.com'   // Ссылка на страницу дома (необязательно)
  },

  // другие участки...
}

Пояснения к параметрам

  • name — название или номер участка. Обязательный параметр. Обычно совпадает с номером участка на карте.
  • status — статус участка. Используйте только предопределённые переменные статусов (см. раздел ниже). От статуса зависит цвет участка на карте и отображение некоторых блоков в балуне. Если параметр не указан — используется STATUS_FREE.
  • area — площадь участка в сотках (число). Отображается в блоке площади участка. Если указана и при этом не задана price, стоимость участка может быть рассчитана автоматически.
  • price — цена участка в рублях (число). Если не указана, цена вычисляется автоматически: price = area × DEFAULT_AREA_PRICE.
  • houseArea — площадь дома в м² (число). Если не указана, блок с информацией о доме (.house) может быть скрыт.
  • housePrice — цена дома в рублях (число). Если не указана, но указана houseArea, стоимость вычисляется автоматически: housePrice = houseArea × DEFAULT_HOUSE_PRICE.
  • totalPrice — общая стоимость участка и дома (число). Если не указана, вычисляется автоматически как сумма: price + housePrice, если они известны.
  • img — имя файла изображения дома (строка). Указывается только имя файла, например: house-180.jpg.
    Изображения должны находиться в папке img/. Если параметр не указан, используется изображение по умолчанию.
  • house-link — ссылка на страницу дома (строка). Если параметр указан, появляется кнопка «Подробнее о доме», которая ведёт на эту страницу. Если параметр отсутствует — кнопка скрывается.

Статусы участка

Чтобы избежать ошибок в написании, используйте в параметре status переменные из кода. Ниже перечислены предопределённые переменные, которые доступны в вашем скрипте:

const STATUS_FREE = 'Свободный участок';
const STATUS_HOUSE_READY = 'Участок с готовым домом';
const STATUS_HOUSE_BUILDING = 'Участок со строящимся домом';
const STATUS_RESERVED = 'Забронированный участок';
const STATUS_SOLD = 'Проданный участок';

Пример правильного указания статуса в locationData:

locationData = {
  'path12345': {
    name: 'Участок 12',
    status: STATUS_HOUSE_READY,
    area: 1800,
    price: 9500000,
    houseArea: 160,
    housePrice: 22000000,
    img: 'plan-image12.jpg'
  }
}

Пример полного объекта locationData

const locationData = {
  'path62461': {
    name: 'Участок 3665',
    status: STATUS_FREE,
    area: 1669,
    price: 8900000,
    houseArea: 150,
    housePrice: 23500000,
    img: 'plan-image1.jpg'
  },
  'path62465': {
    name: 'Участок 3277',
    status: STATUS_SOLD,
    area: 1132,
    price: 6000000,
    houseArea: 140,
    housePrice: 21000000
  },
  'p1-1': {
    name: '1.1',
    status: STATUS_HOUSE_BUILDING,
    area: 1532.24,
    price: 7200000
    // houseArea и housePrice будут использованы значения по умолчанию
  }
};

Советы

  • Всегда используйте переменные для status, чтобы избежать опечаток.
  • Если параметр не нужен или неизвестен, его можно опустить.
  • Цены и площади указывайте в числовом формате (без пробелов и дополнительных символов).
  • Все изображения должны лежать в папке img/ вашего проекта, и указывайте только имя файла в параметре img.
